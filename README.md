# Sistema de Chamados (Help Desk)

Projeto simples desenvolvido para simular um sistema de suporte técnico.

## Funcionalidades
- Abertura de chamados
- Listagem de chamados
- Marcação como resolvido

## Objetivo
Praticar lógica de programação e representar fluxo de atendimento técnico.

## Tecnologias
- HTML
- CSS
- JavaScript
